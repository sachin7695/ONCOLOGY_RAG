# ONCOLOGY_RAG
Medical ONCOLOGY RAG App For Patients

SETUP
$ python ingest.py(vector_db created) </br>
$ docker pull qdrant/qdrant (creating docker image)</br>
$ docker run -p 6333:6333 (on cmd for docker server)</br>
$ uvicorn RAG:APP</br>
